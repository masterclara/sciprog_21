This is the folder for Scientific Programming Concepts - Practical 5

It contains the following source code file(s):
scanf.c		- source code to request user input for one integer and one double variable 
fibonacci.c	- creates program to display the Fibonacci series up to n (user input) using arrays
fibonacci_v2.c	- modification of fibonacci.c using pointers (following practical 5 run through)
hyperarctan.c	- creates program to calculate hyperbolic arc tangent through Maclaurin series and natural logarithms and compares the accuracy of both methods
