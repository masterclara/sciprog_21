Here you will find the C source code files for Scientific Programming Concepts - Practical 2 on 28/09/2021.

Conversion.c 	- Converts a number from a 4-byte integer to a 4-byte float and binary number. Functionality extended to count and return the number of binary digits.
Sum.c		- Highlights limited accuracy of floating point numbers by addition of sums using for loops. 

They have been checked and can be run through sciprog. Instructions to compile the code were added where neccessary.
